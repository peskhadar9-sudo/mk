# Sida Loo Galiyo Mashruuca (Project-ka) GitHub - Tallaabo-Tallaabo

Halkan waxaa ku qoran habka ugu fudud ee aad ku shubi karto mashruucaaga GitHub maadaama barnaamijka Git uusan ku rakibnayn kombuyuutarkaaga. 

Waxaan kuu diyaariyay fayl ZIP ah oo nadiif ah oo ay ku jiraan dhammaan koodhka, sawirka Dashboard-ka, iyo README.md oo sax ah.

---

## 🛠️ Tallaabooyinka Upload-ka (Soo-gelinta) ee Browser-ka:

Maadaama aad horey ugu jirtay koontadaada GitHub ee browser-ka (sida ka muuqata sawirka aad ii soo dirtay), raac tallaabooyinkan fudud:

### Tallaabada 1: Soo bixi Faylka ZIP-ka ah
1. Tag galkaaga **Downloads** ee kombuyuutarkaaga.
2. Waxaad ku arki doontaa fayl la yiraahdo: `AI_Personal_Finance_Advisor_System.zip`.
3. Midigta ku dhufo (Right-click) faylkaas, ka dibna dooro **"Extract All..."** (Soo saar dhammaan) si aad u hesho gal (folder) caadi ah oo ay ku jiraan faylashu.

### Tallaabada 2: Tag Boggaaga GitHub
1. Fur internet-ka (Chrome ama browser-ka aad isticmaasho).
2. Tag bogga keydkaaga (repository): [https://github.com/peskhadar9-sudo/khadarmk](https://github.com/peskhadar9-sudo/khadarmk).

### Tallaabada 3: Shubo Faylasha (Drag and Drop)
1. Bogga GitHub ee maran, waxaad ku arki doontaa qoraal yar oo buluug ah oo oranaya: **"uploading an existing file"** (soo geli fayl jira). Guji qoraalkaas.
2. Hadda, fur galkii aad hadda "unzip" (extract) ku samaysay ee kombuyuutarkaaga ku jiray.
3. Dooro dhammaan faylasha iyo galka ku dhex jira (sida `src`, `public`, `README.md`, `screenshot.png`, `package.json`, iwm.).
4. **Halkan ka taxadar:** Ha dooran galka `node_modules` ama `dist` sababtoo ah aad ayay u culus yihiin (koodhka ZIP-ka ee aan kuu sameeyay horey ayaa looga reebay, markaa kaliya faylasha ku dhex jira galkii aad unzip-ka ku samaysay soo wada jiid).
5. Ku soo tuur (Drag & Drop) daaqadda internet-ka ee GitHub halka ay ku qoran tahay *"Drag files here to add them to your repository"*.

### Tallaabada 4: Keydi Isbeddelada (Commit)
1. Marka ay faylashu wada upload-garoobaan (wada gaba-gaboobaan), hoos ugu dego bogga.
2. Waxaad arki doontaa meel ay ku qoran tahay **"Commit changes"** (Xaqiiji isbeddelada).
3. Guji badhanka cagaaran ee **"Commit changes"**.

**Hambalyo!** Mashruucaagii wuxuu hadda si toos ah ugu jiraa GitHub oo diyaar ah, sawirka dashboard-kana wuxuu si toos ah ugu soo baxayaa bogga hore.
